圖像標籤
<img> (<img src="圖像URL"/>) src為必須屬性 src用於指定圖像文件的路徑和文件名)
<img src="" alt=""/>  alt 替換文本 圖像顯示不出來時用此內文字替換
<img src="" alt="" title=""/> title 提示文本 屬標放到圖像上,提示的文字

<img src="" alt="" title="" width=""/> width 給圖像設定寬度
			    height=""/> height 給圖像設定高度
通常設定寬高其中一個,另一個便會等比例縮放

			    border=""/> border 給圖像設定邊框
屬性與屬性間皆以空格分開

(重點) 超鏈結標籤
<a>
語法: <a href="跳轉目標(url)" target="目標窗口的彈出方式"> 文本或圖像 </a>
target屬性默認值為_self(當前窗口打開葉面) _blank為在新窗口中打開方式

內部鏈接不須加http, 直接打檔案名
空鏈接:<a href="#">空</a>
下載鏈接:href中地址鏈接的是 文件 .exe 或者是zip 等壓縮包形式
錨點鏈接:點我們典籍鏈接,可以快速定位到頁面中的某個位置
	 ．在鏈接文本的href屬性中,設置屬性值為 #名字 的形式,如<a href="#two">第二集</a>
	 ．找到目標位置標籤,裡面添加一個id屬性=剛才的名字,如:<h3 id="two">第二集介紹</h3>

表格
<table>
	<tr>
		<td>單元格內的文字</td>
		...
	</tr>
	...
</table>


1.<table></table> 用於定義表格標籤
2.<tr></tr> 定義表格中的行,必須套在<table></table>
3.<td></td> 定義表格中的單元格,必須套在<tr></tr>
4.字母td指表格數據(table data), 及數據單元格的內容

<table>
	<tr>
		<th>表頭單元格</th>  (字會加粗,居中)
	</tr>
</table>


表格屬性: 不常使用,通常通過css設置
(<table>中屬性)
align		left center right	規定表格相對周圍元素的對其方式
border		1 或 ""			規定表格單元是否有邊框,默認""無邊框
cellpadding	像素值			規定單元邊延與其內容之間的空白,默認1
cellspacing	像素質			規定線寬, 默認2
width		像素質或百分比		表格寬度
height

列表標籤

列表:布局
表格:顯示數據

1.無序列表

以<ul>表示無序列表 而列表項以<li>標籤定義
<ul>中只能放<li>標籤
<li>中可放任何元素

2.有序列表

<ol>表有序列表 而列表項以<li>標籤定義
<ol>中只能套<li> 
<li>可容所有元素

3.自定義列表

<dl>表示自訂義列表 內涵<dt>定義項目/名字 與 <dd>描述每個項目
<dl>中只能包含<dt>和<dd>
一個<dl>中可有多個<dt><dd>

表單標籤
包含:表單域 表單控鍵 提示訊息

1.表單域
<form>用於定義表單域

<form action="url地址" method="提交方式" name="表單域名稱">
	各種表單元入控件
</form>

action		url地址		指定接收處理表單數據的服務氣程序的url地址
method		get/post	設置表單數據的提交方式
name		名稱		表單名稱,區分同業面中的多個表單域

2.表單控件

	(1)input輸入表單元素

	<input type="屬性值" /> (單標籤)
	type屬性值: button(普通按鈕) checkbox(多選框) file(上船文件使用的) hidden image password radio(單選框按鈕,按鈕必須有相同的name才可以實現多選一) reset(重製按鈕(重寫表單)) submit(定義提交按鈕) text(單純輸入)
	

	input其他屬性:name(定義input元素名稱) value(規定input元素的值(提示(亦是默認值))) checked(規定此input元素首次加載時應當被選中) maxlength(規定輸入字串的最大長度)
	name value是每個表單元素都有的屬性值,主要給後台人員使用
	單選和複選框都要有相同的name值
	checked屬性主要針對單選按鈕及複選框

	<label for="id">男</label>
	<input type="radio" name="sex" id="sex"/>
	(用於綁訂一個表單元素,當點擊label內文本時,瀏覽器就會自動選擇對應的表單元素上,增加用戶體驗)

	(2)select下拉表單元素

	<select>
		<option>選像1</option>
		<option>     </option>
		<option>     </option>
		...
	</select>
	在option中定義selected="selected" 時,當前選像及為默認選中
	

	(3)textarea文本域元素
	
	用於定義多行文本輸入,可以輸入更多文字,常見留言板,評論
	<textarea rows="3" cols="20">
		文本內容
	</textarea>
